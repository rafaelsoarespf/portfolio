Aurora CSS

Um framework CSS moderno focado em elegância, acessibilidade e design responsivo.

Aurora CSS foi projetado com base em uma arquitetura orientada a temas, permitindo que as interfaces adaptem sua identidade visual sem alterar sua estrutura. Componentes, utilitários e animações são desenvolvidos para funcionar de maneira consistente em múltiplos temas, possibilitando a criação de sistemas de design escaláveis e de fácil manutenção.
